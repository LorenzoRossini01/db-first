# Esercizio di oggi: Primo DB

nome repo: db-first

Modellizzare la struttura di una tabella per memorizzare tutti i dati riguardanti delle auto usate messe in vendita da un concessionario

## Svolgimento

Dati da raccogliere:

- id
- modello
- marca
- anno di produzione
- km percorsi
- ultima revisione
- incidenti subiti
- precedente possessore
- posteggio
- cambio
- carburante
- prezzo
- trazione
- equipaggiamento
- tipo di carrozzeria
- potenza
- numero di portiere
- numero sedili
- colore
